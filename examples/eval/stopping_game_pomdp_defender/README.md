# Policy Evaluation

This directory contains example scripts for policy evaluation for the stopping POMDP from [https://ieeexplore.ieee.org/document/9779345](https://ieeexplore.ieee.org/document/9779345).

## Commands

To run a script, execute:
```bash
python <script_name>
```
