# SARSA for MDP

This directory contains example scripts for optimizing attacker policies using SARSA for the MDP from [https://limmen.dev/assets/papers/icml_ml4cyber_Hammar_Stadler_final_24_june_2022.pdf](https://limmen.dev/assets/papers/icml_ml4cyber_Hammar_Stadler_final_24_june_2022.pdf).

## Commands

To run a script, execute:
```bash
python <script_name>
```
