# Threshold-Fictitious Self-Play for Partially Observed Stochastic Game

This directory contains example scripts for computing Nash equilibria using T-FP for the stopping game from [https://limmen.dev/assets/papers/icml_ml4cyber_Hammar_Stadler_final_24_june_2022.pdf](https://limmen.dev/assets/papers/icml_ml4cyber_Hammar_Stadler_final_24_june_2022.pdf)

## Commands

To run a script, execute:
```bash
python <script_name>
```
